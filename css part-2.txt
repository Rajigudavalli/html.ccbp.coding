CSS Text Properties:

1. Color
The CSS color property specifies the color of the text.
 
    .main-heading {      ------ syntax
  color: blue;
}
.paragraph {
  color: grey;
}

CSS Background Properties :

2. Background Color

The CSS background-color property specifies the background color of an HTML element.

.card {                     ------ syntax
  background-color: lightblue;
}
 ------ example problem---------
